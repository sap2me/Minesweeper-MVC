Minesweeper-MVC
=======
Minesweeper game written in `python3`,`PyQT5` using Model View Controller design pattern and test-driven development technology. You can dowload [exe](https://github.com/sap2me/Minesweeper-MVC/releases/download/1.00/minesweeper_exe.rar "onefile exe") file.
